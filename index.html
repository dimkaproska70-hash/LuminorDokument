Установка (загрузчик)

Подключите библиотеку через loadstring, указав прямую ссылку на raw‑версию файла с GitHub. 

local LuminorGui = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()

💡 Требуется доступ к game:HttpGet. Библиотека совместима с большинством исполнителей.

Создание главного окна

CreateWindow(title, uiName, watermark, themeName) – возвращает объект окна.
Параметры:
• title – заголовок (плавная пульсация цвета темы и белого)
• uiName – имя ScreenGui (опционально)
• watermark – текст внизу (по умолч. "By DADILK")
• themeName – тема оформления: "classic", "red", "mint", "orange" (или на русском: "классический", "красный", "мятный", "оранжевый") 

local Window = LuminorGui:CreateWindow("Моё меню", "MyUI", "Пример", "mint")

Окно имеет вращающуюся радужную обводку (градиент темы и темноты). При нажатии на свернуть (желтый кружок) срабатывает анимация волны, а GUI превращается в компактную верхнюю панель, окрашенную под цвет выбранной темы.

Создание вкладок (Tabs)

Window:CreateTab(iconText) – добавляет вкладку в боковое меню.
iconText – любой текст или эмодзи.
Возвращает объект для добавления элементов. 

local Tab1 = Window:CreateTab("📁 Главная")
local Tab2 = Window:CreateTab("⚙️ Настройки")

Вкладки переключаются по клику, содержимое каждой в отдельном ScrollingFrame.

Переключатели (Toggle)

Tab:CreateToggle(text, neonColor, defaultState, callback)
• text – подпись
• neonColor – цвет подсветки (Color3)
• defaultState – true / false
• callback – функция, принимает новое состояние

"Purple"
"Red"
"Green"
"Blue"
"Yellow"
"Orange"
"Pink"
"White"
"Cyan"
local Purple = Color3.fromRGB(192, 132, 252)

Tab1:CreateToggle("Включить режим", Purple, false, function(state)
    print("Состояние:", state)
end)

При клике меняет цвет текста, показывает неоновую линию снизу и вызывает колбэк.

Обычные кнопки (Button)

Tab:CreateButton(text, callback)
• text – надпись на кнопке
• callback – функция при клике 

Tab1:CreateButton("Нажми меня", function()
    print("Кнопка нажата!")
end)

Кнопки не имеют состояния, а при нажатии запускают эффект круговой расходящейся волны.

Готовый скрипт

Полный рабочий код с мятным оформлением интерфейса. Включает две вкладки с интерактивными элементами и полностью готов для использования в Roblox. 

-- Загружаем библиотеку
local LuminorGui = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()

-- Создаём окно с темой "mint" (также доступны: "classic", "red", "orange")
local Window = LuminorGui:CreateWindow("Luminor", "LuminorUI", "By DADILK", "mint")

-- Вкладки
local Tab1 = Window:CreateTab("📁 Главная")
local Tab2 = Window:CreateTab("⚙️ Настройки")

-- Цвета для переключателей
local MintColor = Color3.fromRGB(85, 255, 170)
local Orange = Color3.fromRGB(255, 170, 0)

-- Tab 1
Tab1:CreateToggle("Быстрый бег", MintColor, false, function(s) print("Состояние бега:", s) end)
Tab1:CreateButton("Убить всех", function() print("Выполнено!") end)

-- Tab 2
Tab2:CreateToggle("Оранжевый тоггл", Orange, false, function(s) print("Оранжевый:", s) end)
Tab2:CreateButton("Сброс настроек", function() print("Настройки сброшены") end)

Скопируйте и вставьте в свой чит-исполнитель – интерфейс со всеми обновленными анимациями появится сразу.
