Загрузка библиотеки

Скопируйте этот код в начало вашего скрипта, чтобы подключить LuminorLib.

LUA 
local LuminorLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()
Далее: Создание Окна

1. Создание окна и темы

Метод CreateWindow принимает пять параметров. Последний — имя фона (или "none"). Поддерживаются темы: "classic" (светлая), "rainbow" (переливающаяся), "purple", "red" и т.д.

LUA 
local Window = LuminorLib:CreateWindow(
    "Мой скрипт",    -- Заголовок
    "MyUI",          -- Имя ScreenGui
    "by DADILK",     -- Ватермарка
    "rainbow",       -- Тема (classic/rainbow/purple/red/...)
    "Pan1"           -- Фон (имя файла или "none")
)
Назад Далее: Вкладки

2. Вкладки (Tabs)

Иконки указываются без расширения. Пустой текст — только иконка по центру.

LUA 
-- Обычная вкладка
local MainTab = Window:CreateTab("Главная", "home")
-- Компактная вкладка с иконкой
local SettingsTab = Window:CreateTab("", "settings")
Назад Далее: Переключатели

3. Переключатели (Toggles)

Порядок параметров: CreateToggle(текст, состояние_по_умолчанию, неоновый_цвет, иконка, callback).

LUA 
MainTab:CreateToggle("Авто-Фарм", false, "green", "farm", function(state)
    print("Авто-Фарм:", state)
end)

MainTab:CreateToggle("", false, "purple", "eye", function(state)
    print("ESP:", state)
end)
Назад Далее: Кнопки

4. Интерактивные Кнопки

Кнопки с эффектом волны. Иконка задаётся именем без расширения.

LUA 
MainTab:CreateButton("Телепорт к боссу", "teleport", function()
    print("Телепортация...")
end)

MainTab:CreateButton("", "trash", function()
    print("Конфиг очищен!")
end)
Назад Далее: Слайдеры

5. Слайдеры (Sliders)

Теперь слайдер принимает цвет и иконку: CreateSlider(текст, мин, макс, по_умолчанию, неоновый_цвет, иконка, callback).

LUA 
MainTab:CreateSlider("Громкость", 0, 100, 50, "green", "", function(v)
    print("Громкость:", v)
end)

MainTab:CreateSlider("", 1, 10, 5, nil, "", function(v)
    print("Уровень:", v)
end)
Назад Далее: Dropdown

6. Выпадающий список (Dropdown)

Создаёт раскрывающийся список с анимацией: CreateDropdown(текст, список_вариантов, по_умолчанию, неоновый_цвет, иконка, callback).

LUA 
MainTab:CreateDropdown(
    "Выбрать игрока",               -- текст
    {"Player1", "Player2", "Player3"}, -- список
    "Player1",                      -- по умолчанию
    "cyan",                         -- цвет выбранного
    "",                             -- иконка (или nil)
    function(selected)              -- callback
        print("Выбран:", selected)
    end
)
Назад Далее: Markdown

7. Markdown / RichText

Добавьте форматированный текст с помощью CreateMarkdown(текст). Поддерживаются теги Roblox RichText.

LUA 
local myLabel = MainTab:CreateMarkdown(
    "<b>Привет</b> <font color='#c084fc'>LuminorLib</font>!<br>Версия 1.2.1"
)
-- можно потом изменить текст: myLabel.Text = "..."
Назад Далее: ScrollingBox

8. Прокручиваемый контейнер (ScrollingBox)

Создаёт область с внутренним скроллом: CreateScrollingBox(высота). Возвращает ScrollingFrame, в который можно добавлять элементы.

LUA 
local box = MainTab:CreateScrollingBox(200) -- высота 200 пикселей
-- box это ScrollingFrame, можно добавлять кнопки/текст и т.д.
local btn = Instance.new("TextButton")
btn.Size = UDim2.new(1, 0, 0, 30)
btn.Text = "Кнопка внутри"
btn.Parent = box
Назад Финальный шаг: Полный Шаблон

9. Готовый шаблон скрипта

Этот пример использует все доступные функции библиотеки: окно, вкладки, тогглы, кнопки, слайдер, dropdown, markdown и scrolling box.

ПОЛНЫЙ СКРИПТ LUA 
local LuminorLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()

local Window = LuminorLib:CreateWindow("Luminor Premium", "UI", "by DADILK", "rainbow", "Pan1")

local MainTab = Window:CreateTab("Главная", "home")
local CombatTab = Window:CreateTab("Бой", "sword")
local VisualsTab = Window:CreateTab("Визуалы", "eye")
local SettingsTab = Window:CreateTab("Настройки", "settings")

-- Тогглы
MainTab:CreateToggle("Авто-Фарм", false, "green", "farm", function(state)
    if state then print("Фарм запущен") end
end)

CombatTab:CreateToggle("Аимбот", false, "purple", "aim", function(state)
    print("Аимбот:", state)
end)

VisualsTab:CreateToggle("ESP", false, "cyan", "user", function(state)
    print("ESP:", state)
end)

-- Кнопки
MainTab:CreateButton("Телепорт к боссу", "teleport", function()
    print("Телепорт...")
end)

SettingsTab:CreateButton("Сохранить конфиг", "save", function()
    writefile("config.json", game:GetService("HttpService"):JSONEncode({volume = 75}))
end)

-- Слайдер
SettingsTab:CreateSlider("Громкость", 0, 100, 75, "green", "", function(v)
    print("Громкость:", v)
end)

-- Dropdown
MainTab:CreateDropdown("Выбрать игрока", {"Player1", "Player2", "Player3"}, "Player1", "cyan", "", function(selected)
    print("Выбран:", selected)
end)

-- Markdown
MainTab:CreateMarkdown("<b>LuminorLib v1.2.1</b> <font color='#c084fc'>активна</font>")

-- ScrollingBox
local box = MainTab:CreateScrollingBox(150)
local btn = Instance.new("TextButton")
btn.Size = UDim2.new(1, 0, 0, 30)
btn.Text = "Кнопка внутри скролла"
btn.Parent = box
