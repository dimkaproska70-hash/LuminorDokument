Установка (загрузчик)

Подключите библиотеку через loadstring, указав прямую ссылку на raw‑версию файла с GitHub. 

local LuminorGui = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()

💡 Требуется доступ к game:HttpGet. Библиотека совместима с большинством исполнителей.

Создание главного окна

CreateWindow(title, uiName, watermark) – возвращает объект окна.
Параметры:
• title – заголовок (неоновая анимация)
• uiName – имя ScreenGui (опционально)
• watermark – текст внизу (по умолч. "By DADILK") 

local Window = LuminorGui.CreateWindow("Моё меню", "MyUI", "Пример")

Окно имеет кнопки свернуть / закрыть и изменение размера перетаскиванием за угол.

Создание вкладок (Tabs)

Window:CreateTab(iconText) – добавляет вкладку в боковое меню.
iconText – любой текст или эмодзи.
Возвращает объект для добавления элементов. 

local Tab1 = Window:CreateTab("📁 Главная")
local Tab2 = Window:CreateTab("⚙️ Настройки")

Вкладки переключаются по клику, содержимое каждой в отдельном ScrollingFrame.

Переключатели (Toggle)

Tab:CreateToggle(text, neonColor, defaultState, callback)
• text – подпись
• neonColor – цвет подсветки (Color3)
• defaultState – true / false
• callback – функция, принимает новое состояние

local Purple = Color3.fromRGB(192, 132, 252)

Tab1:CreateToggle("Включить режим", Purple, false, function(state)
    print("Состояние:", state)
end)

При клике меняет цвет текста, показывает неоновую линию снизу и вызывает колбэк.
👆 Кликните на цветной квадрат выше – цвет демо-переключателя изменится.

Обычные кнопки (Button)

Tab:CreateButton(text, callback)
• text – надпись на кнопке
• callback – функция при клике 

Tab1:CreateButton("Нажми меня", function()
    print("Кнопка нажата!")
end)

Кнопки не имеют состояния, просто выполняют действие.

Готовый скрипт

Полный рабочий код с тремя вкладками, в каждой тоггл и кнопка. Цвета подобраны для наглядности. Убедитесь, что ваша версия библиотеки поддерживает CreateButton. 

-- Загружаем библиотеку
local LuminorGui = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()

-- Создаём окно
local Window = LuminorGui.CreateWindow("test")

-- Вкладки
local Tab1 = Window:CreateTab("test 1")
local Tab2 = Window:CreateTab("test 2")
local Tab3 = Window:CreateTab("test 3")

-- Цвета
local Orange = Color3.fromRGB(255, 170, 40)
local White  = Color3.fromRGB(255, 255, 255)
local Purple = Color3.fromRGB(192, 132, 252)

-- Tab1
Tab1:CreateToggle("test", Orange, false, function(s) print("Оранжевый:", s) end)
Tab1:CreateButton("test", function() print("Кнопка Tab1") end)

-- Tab2
Tab2:CreateToggle("test", White, false, function(s) print("Белый:", s) end)
Tab2:CreateButton("test", function() print("Кнопка Tab2") end)

-- Tab3
Tab3:CreateToggle("test", Purple, false, function(s) print("Фиолетовый:", s) end)
Tab3:CreateButton("test", function() print("Кнопка Tab3") end)

Скопируйте и вставьте в свой скрипт – интерфейс появится сразу. Если CreateButton отсутствует, обновите библиотеку или замените на CreateToggle с фиктивным состоянием.
