6. Маркдаун / RichText

Для вывода форматированного текста используйте CreateMarkdown(text). Поддерживаются стандартные теги Roblox RichText: <b>, <i>, <u>, <font color="...">, <br> и другие.

LUA 
-- Синтаксис: CreateMarkdown(richText)
MainTab:CreateMarkdown(
    "<b>Добро пожаловать</b> в <font color='#c084fc'>LuminorLib</font>!<br>" ..
    "<i>Версия 1.2.1</i> — <u>обновлённый интерфейс</u>."
)

-- Можно передавать обычный текст без тегов
MainTab:CreateMarkdown("Простой текст тоже работает.")

Маркдаун-блок автоматически растягивается по ширине контейнера и поддерживает перенос строк.
