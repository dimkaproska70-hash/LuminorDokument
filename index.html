Введение

Добро пожаловать в обновленную версию библиотеки LuminorLib v1.1. Теперь интерфейс полностью поддерживает кастомную графику, анимированные задние фоны (.mp4/.webm) и умную систему иконок, которые автоматически скачиваются с вашего репозитория и адаптируются под размеры кнопок.

Для корректной работы фонов и иконок ваш эксплоит (executor) должен обладать поддержкой функций работы с файловой системой (writefile, makefolder) и генерации локальных ассетов (getcustomasset).

Создание окна и задний фон

Теперь в метод CreateWindow добавлен пятый (необязательный) параметр — bgName. Вы можете передать имя файла, и библиотека автоматически скачает его из вашей папки Background на GitHub.

Доступные заготовки фонов:

Pan1.mp4 (Анимация)
Pan2.png (Статичный)
Pan3.png (Статичный)
Pan4.png (Статичный)
Pan5.png (Статичный)

Пример создания окна с анимированным фоном:

local Window = LuminorLib:CreateWindow(
    "Luminor Premium", -- Заголовок
    "Luminor_UI",       -- Имя ScreenGui
    "by DADILK",        -- Ватермарка
    "Classic",          -- Цветовая тема
    "Pan1.mp4"          -- Название файла фона (или "Pan3.png")
)

Создание Вкладок (Tabs)

Вкладки теперь принимают параметр iconName. Иконка центрируется вместе с текстом вкладки. Если передать пустую строку в качестве текста, вкладка отобразит только саму иконку по центру.

Пример добавления вкладок:

-- Вкладка с текстом и иконкой
local MainTab = Window:CreateTab("Главная", "home.ico")

-- Вкладка только с иконкой шестеренки
local SettingsTab = Window:CreateTab("", "settings.ico")

Переключатели (Toggles)

Переключатель теперь поддерживает умную иконку, которая динамически меняет цвет вместе с текстом (загорается выбранным неоновым цветом при активации).

Пример создания:

-- Сигнатура: CreateToggle(text, neonColor, defaultState, iconName, callback)
MainTab:CreateToggle("Авто-Фарм", "green", false, "farm.ico", function(state)
    print("Авто-фарм равен: " .. tostring(state))
end)

-- Тоггл без текста (только иконка глаза по центру)
MainTab:CreateToggle("", "purple", false, "eye.ico", function(state)
    print("Глаз нажат: " .. tostring(state))
end)

Кнопки (Buttons)

Кнопки также обзавелись поддержкой иконок, которые размещаются справа от текста или по центру, если текст отсутствует. Оснащены эффектом радиального волны (ripple effect) при нажатии.

Пример создания:

-- Сигнатура: CreateButton(text, iconName, callback)
MainTab:CreateButton("Телепорт к боссу", "teleport.ico", function()
    print("Телепортация...")
end)

-- Кнопка без текста (только центрированная иконка корзины)
MainTab:CreateButton("", "trash.ico", function()
    print("Очистка завершена!")
end)

Готовый пример использования

Просто скопируйте и запустите этот пример в вашей любимой утилите для исполнения скриптов (например, Wave, Synapse, Solara и др.): 

local LuminorLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()

-- 1. Создаем красивое окно с анимированным фоном Pan1
local Window = LuminorLib:CreateWindow("Luminor Hub", "LuminorUI", "By DADILK", "classic", "Pan1.mp4")

-- 2. Добавляем вкладки с иконками
local MainTab = Window:CreateTab("Основные", "home.ico")
local ConfigTab = Window:CreateTab("Настройки", "settings.ico")

-- 3. Добавляем интерактивные тогглы
MainTab:CreateToggle("Включить Аимбот", "purple", false, "aim.ico", function(state)
    print("Aimbot:", state)
end)

MainTab:CreateToggle("Быстрый бег", "cyan", true, "speed.ico", function(state)
    print("Speedhack:", state)
end)

-- 4. Добавляем кнопки
ConfigTab:CreateButton("Сохранить конфиг", "save.ico", function()
    print("Конфиг успешно сохранен!")
end)

ConfigTab:CreateButton("Сбросить все", "trash.ico", function()
    print("Конфигурация сброшена.")
end)
