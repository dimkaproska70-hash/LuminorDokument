Документация

Шаг 1: Загрузка библиотеки

Подключите библиотеку LuminorLib через loadstring, указав прямую ссылку на raw‑версию скрипта с GitHub. 

local LuminorLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()

💡 Убедитесь, что ваш исполнитель скриптов (executor) поддерживает функцию game:HttpGet.

Шаг 2: Создание главного окна

Используйте функцию CreateWindow, чтобы создать каркас интерфейса. Оно появится после красивой заставки с эффектом Pop-up.
Параметры:
• titleText – заголовок
• uiName – техническое имя интерфейса
• watermarkText – подпись в левом нижнем углу 

local Window = LuminorLib:CreateWindow("Панель Управления", "MyCoolUI", "Создано с LuminorLib")

Готовое окно поддерживает перемещение за фон, изменение размера за правый нижний угол и сворачивание.

Шаг 3: Группировка по вкладкам

Вкладки (Tabs) располагаются в левой части меню. Под активной вкладкой всегда горит белая неоновая полоска. Метод CreateTab возвращает объект, к которому привязываются кнопки. 

local Tab1 = Window:CreateTab("⚔️ Функции")
local Tab2 = Window:CreateTab("👁️ ESP")

Шаг 4: Переключатели (Toggles)

Тоггл запоминает состояние (Вкл/Выкл) и зажигает цветную неоновую полоску. Отлично подходит для активации читов, автофарма и ESP. 

local Purple = Color3.fromRGB(192, 132, 252)

Tab1:CreateToggle("Авто-Фарм", Purple, false, function(state)
    if state then
        print("Авто-Фарм запущен!")
    else
        print("Авто-Фарм остановлен!")
    end
end)

defaultState (в примере false) определяет, включен ли тоггл изначально при запуске.

Шаг 5: Обычные кнопки (Buttons)

Кнопка (Button) используется для разовых действий: телепорт, спавн предмета или сброс настроек. У кнопок нет состояния (Вкл/Выкл), но при нажатии воспроизводится красивая анимация расходящейся волны (ripple-эффект). 

Tab1:CreateButton("Телепорт на спавн", function()
    print("Кнопка нажата! Выполняю действие...")
end)

💡 В отличие от тогглов, для кнопок не нужно указывать цвет, они имеют стандартный строгий вид.

Готовое меню

Вот полный рабочий шаблон для создания скрипта, который включает и тогглы, и обычные кнопки. 

-- 1. Загружаем библиотеку
local LuminorLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()

-- 2. Создаём главное окно
local Window = LuminorLib:CreateWindow("Mega Script", "MyUI", "LN V1.0")

-- 3. Добавляем вкладки
local TabMain = Window:CreateTab("⚔️ Основное")
local TabPlayer = Window:CreateTab("🏃 Игрок")

-- Настраиваем цвета для переключателей
local Orange = Color3.fromRGB(255, 170, 40)
local Purple = Color3.fromRGB(192, 132, 252)

-- 4. Наполняем вкладку "Основное" (Тогглы)
TabMain:CreateToggle("Авто-Атака", Orange, false, function(s) 
    print("Авто-Атака:", s) 
end)

-- Добавляем обычную кнопку во вкладку "Основное"
TabMain:CreateButton("Выдать меч", function() 
    print("Меч выдан!") 
end)

-- 5. Наполняем вкладку "Игрок"
TabPlayer:CreateToggle("Подсветка (ESP)", Purple, false, function(s) 
    print("ВХ:", s) 
end)

TabPlayer:CreateButton("Телепорт на базу", function() 
    print("Телепорт выполнен!") 
end)
