Загрузка библиотеки

Скопируйте этот код в начало вашего скрипта, чтобы подключить LuminorLib.

LUA 
local LuminorLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()
Далее: Создание Окна

1. Создание окна, фоны и Platoboost

Метод CreateWindow принимает таблицу опций. Можно задать фон по имени без расширения и скрытые ID/API для Platoboost.

LUA 
local Window = LuminorLib:CreateWindow(
    "Luminor Premium",   -- Заголовок
    "Luminor_UI",        -- Имя ScreenGui
    "by DADILK",         -- Ватермарка
    "Classic",           -- Цветовая тема
    "Pan1",              -- Фон (автопоиск Pan1.mp4 / Pan1.png)
    {
        keySystem = true,               -- Включить ключ-систему
        platoboostId = "123456",        -- Ваш ID проекта
        platoboostApiKey = "key_abcde"  -- Ваш API ключ
    }
)
Назад Далее: Вкладки

2. Вкладки (Tabs)

Иконки теперь указываются без расширения. Пустой текст — только иконка по центру.

LUA 
-- Обычная вкладка
local MainTab = Window:CreateTab("Главная", "home")
-- Компактная вкладка с иконкой
local SettingsTab = Window:CreateTab("", "settings")
Назад Далее: Переключатели

3. Переключатели (Toggles)

Иконка автоматически загорается неоновым цветом при активации.

LUA 
MainTab:CreateToggle("Авто-Фарм", "green", false, "farm", function(state)
    print("Авто-Фарм: " .. tostring(state))
end)

MainTab:CreateToggle("", "purple", false, "eye", function(state)
    print("ESP: " .. tostring(state))
end)
Назад Далее: Кнопки

4. Интерактивные Кнопки

Кнопки с эффектом волны. Иконка задаётся именем без расширения.

LUA 
MainTab:CreateButton("Телепорт к боссу", "teleport", function()
    print("Телепортация...")
end)

MainTab:CreateButton("", "trash", function()
    print("Конфиг очищен!")
end)
Назад Далее: Слайдеры

5. Слайдеры (Sliders)

Новый элемент с плавным ползунком и отображением значения.

LUA 
MainTab:CreateSlider("Громкость", 0, 100, 50, function(value)
    print("Громкость: " .. value)
end)

MainTab:CreateSlider("", 1, 10, 5, function(value)
    print("Уровень: " .. value)
end)
Назад Далее: Маркдаун

6. Маркдаун / RichText

Вывод форматированного текста с поддержкой RichText-тегов.

LUA 
MainTab:CreateMarkdown(
    "<b>Привет</b> <font color='#c084fc'>LuminorLib</font>!<br>" ..
    "<i>Версия 1.2.1</i> — <u>обновление</u>."
)

MainTab:CreateMarkdown("Обычный текст без тегов.")
Назад Далее: Полный Шаблон

7. Готовый шаблон скрипта

Пример скрипта со всеми элементами, сохранением конфига и ключ-системой.

ПОЛНЫЙ СКРИПТ LUA 
local LuminorLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/dimkaproska70-hash/LuminorGui/main/Lib.lua"))()

local Window = LuminorLib:CreateWindow("Luminor Premium", "UI", "by DADILK", "classic", "Pan1", {
    keySystem = true,
    platoboostId = "123456",
    platoboostApiKey = "your_api_key"
})

local MainTab = Window:CreateTab("Главная", "home")
local CombatTab = Window:CreateTab("Бой", "sword")
local SettingsTab = Window:CreateTab("Настройки", "settings")

MainTab:CreateToggle("Авто-Фарм", "green", false, "farm", function(state)
    if state then print("Фарм запущен") end
end)

MainTab:CreateButton("Телепорт", "teleport", function()
    print("Телепорт...")
end)

MainTab:CreateMarkdown("<b>Готово!</b> <font color='#c084fc'>Добро пожаловать</font>")

CombatTab:CreateToggle("Аимбот", "purple", false, "aim", function(state)
    print("Аимбот:", state)
end)

SettingsTab:CreateSlider("Громкость", 0, 100, 75, function(v)
    print("Громкость:", v)
end)

SettingsTab:CreateButton("Сохранить", "save", function()
    writefile("config.json", game:GetService("HttpService"):JSONEncode({volume = 75}))
end)
